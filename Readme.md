# 🤖 PL Roobotics

A futuristic robotics landing page built using React, Vite, and Tailwind CSS.

## 🚀 Features

- Responsive UI
- Modern robotics design
- Reusable React components
- Smooth animations
- Contact modal
- Tailwind CSS styling

## 🛠️ Tech Stack

- React
- Vite
- Tailwind CSS
- JavaScript (JSX)

## 📂 Structure

```bash
src/
├── assets/
├── components/
├── hooks/
├── App.jsx
├── main.jsx
└── index.css
```

## ⚙️ Run Locally

```bash
npm install
npm run dev
```

## 📦 Build

```bash
npm run build
```
